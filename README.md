# Python Job Scraper

Scraper em Python que coleta vagas do site Fake Python Jobs, extraindo título, empresa, localização e link. Os dados são organizados e exportados para CSV.

## 🚀 Tecnologias utilizadas

- Python
- Requests
- BeautifulSoup (bs4)
- CSV

## 📊 Dados extraídos

- Título da vaga
- Empresa
- Localização
- Link da vaga
